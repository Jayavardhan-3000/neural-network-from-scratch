# Neural Network From Scratch

Built a neural network completely from scratch using only NumPy.
No deep learning frameworks were used.

## Features
- Custom Dense layers
- ReLU activation
- Mean Squared Error loss
- Forward & backward propagation
- Trained on non-linear data (quadratic regression)

## Tech Stack
- Python
- NumPy
- Matplotlib

## Notebook
All implementation details are available in the Jupyter Notebook.
